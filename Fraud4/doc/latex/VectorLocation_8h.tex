Overloading of the stream extraction operator for \doxylink{classVectorLocation}{Vector\+Location} class. This operator first reads from the input stream a line containing an integer n, which corresponds to the number of \doxylink{classLocation}{Location} objects to be read from the input stream. Then, it reads n \doxylink{classLocation}{Location} objects from the input stream, storing them in the provided \doxylink{classVectorLocation}{Vector\+Location} object ({\ttfamily vector}). 

\begin{DoxyNote}{Note}
This operator should remove any \doxylink{classLocation}{Location} previously contained in the provided \doxylink{classVectorLocation}{Vector\+Location} {\ttfamily vector}. 

This operator throws an exception in some error cases (see below). Before throwing the corresponding exception, this operator clears the provided object (it calls to clear() method) to leave it in a consistent state. 
\end{DoxyNote}
