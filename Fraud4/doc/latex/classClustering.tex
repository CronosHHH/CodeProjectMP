\doxysection{Clustering Class Reference}
\hypertarget{classClustering}{}\label{classClustering}\index{Clustering@{Clustering}}


This class is used to obtain a clustering of a given set of locations (\doxylink{classLocation}{Location} objects). A clustering algorithm groups the input locations into K disjoint groups (clusters). The clustering algorithm used in this class is the K-\/\+Means algorithm. Each group will consist of a disjunct subset of the original location set and a representative location (centroid), which is calculated as the vector of the means of the locations in the group. The \doxylink{classClustering_a17977c6467960744c8bc71ffb97aa9fc}{run()} method in this class executes the algorithm. An example of use is the following\+: \doxylink{classVectorLocation}{Vector\+Location} locations; Suppose here code to fill the vector of locations \doxylink{classClustering}{Clustering} clustering; clustering.\+set(locations, 5); clustering.\+run();.

This class is used to obtain a clustering of a given set of locations (\doxylink{classLocation}{Location} objects). A clustering algorithm groups the input locations into K disjoint groups (clusters). The clustering algorithm used in this class is the K-\/\+Means algorithm. \doxysubsubsection{\texorpdfstring{run()}{run()}}
{\footnotesize\ttfamily void Clustering\+::run (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Run the clustering algorithm. The clustering algorithm implemented ~\newline
 in this method is the KMeans algorithm. After running this method the following postcondition should be verified\+: 

Metodo run. Realiza el algoritmo KMeans. Para ello, inicializa clusters aleatorios a las localizaciones. Posteriormente, recalcula los centroides haciendo una media de las localizaciones de cada cluster. Finalmente, reasigna cada localizacion a su centroide mas cercano. Esto se repite hasta que los centroides estan en la posicion perfecta y ninguna localizacion cambia de cluster.

\begin{DoxyPostcond}{Postcondition}
The \+\_\+centroids field should contain the vector of the means of locations in each group 

\+\_\+clusters (a \doxylink{classVectorInt}{Vector\+Int} object) should contain the cluster number ~\newline
 for each location in the vector of locations (\+\_\+locations) 

The \+\_\+is\+Done field should be set to true 

The \+\_\+sum\+WCV field should be set to the sum of the within-\/cluster variances using the claculate\+Sum\+Within\+Cluster\+Variances() method. 

The \+\_\+number\+Iterations field should be set to the number of ~\newline
 iterations used by the KMeans algorithm.
\end{DoxyPostcond}
A brief description of the KMeans algorithm is the following\+:


\begin{DoxyEnumerate}
\item Perform an initial assignment of a cluster to each of the \doxylink{classLocation}{Location} objects. This must be done by initializing the random number generator using srand(seed); Then, for each location i, assign cluster number rand() \% \+\_\+K to that location. Note that this step is done by modifying the \+\_\+clusters field.
\item Repeat while any location changes its cluster number\+:
\begin{DoxyItemize}
\item For each cluster, recalculate its centroid (update \+\_\+centroids field).
\item Assign each location to a cluster (update \+\_\+clusters field). Each location is assigned to the cluster that provides the shortest distance to its centroid.
\end{DoxyItemize}
\end{DoxyEnumerate}
