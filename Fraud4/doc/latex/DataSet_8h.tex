class \mbox{\hyperlink{classDataSet}{Data\+Set}}
\begin{DoxyCompactList}\small\item\em A \doxylink{classDataSet}{Data\+Set} is an object that contains a set of n instances. Each instance contains a vector of m integers that represents the amount of money spent by a person in each one of m locations. This class uses a bidimensional matrix with n rows and m columns to store the values of the set of instances. This class also contains a vector of n labels (\doxylink{classVectorInt}{Vector\+Int} \+\_\+labels) to store the label of each instance. Each label is an integer that defines the class value of the corresponding instance. Finally, this class contains a vector of m \doxylink{classLocation}{Location} objects (\doxylink{classVectorLocation}{Vector\+Location} \+\_\+locations) to store each locations in the \doxylink{classDataSet}{Data\+Set}. In the 2D matrix, each column i is associated with location i in the \+\_\+locations vector. \end{DoxyCompactList}\end{DoxyCompactItemize}

Classifies the instances of the dataset {\ttfamily data\+Set\+To\+Classify} with the KNN algorithm and using the dataset {\ttfamily data\+Set\+Model} as model to classify. Each instance in {\ttfamily data\+Set\+To\+Classify} will be classified by changing its label to the one found by the KNN algorithm.
