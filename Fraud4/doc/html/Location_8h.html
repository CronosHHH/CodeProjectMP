std::istream & operator>> (std::istream &is, Location &location)
 Overloading of the stream extraction operator for Location class. It reads two double values (space separated) from the input stream that are used to set the x and y coordinates of the given Location object. Then, it reads the string that can be formed from the current position of the input stream to the end of the line. This string is then trimmed (see the Trim(string) function) to remove its spaces at the beginning and at the end. The resulting string is used to set the name of the provided location. Note that the name can have several words separated by whitespaces.

void Trim ( std::string &  myString)

Removes spaces and \t characters at the beginning and at the end of the provided string myString. If the provided string myString is empty or contains only spaces or \t characters then myString will contain an empty string after calling to this function.

Note
