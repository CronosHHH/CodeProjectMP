Overloading of the stream extraction operator for VectorInt class.

Note
This operator should remove any previously information contained in the provided VectorInt vector.
This operator throws an exception in some error cases (see below). Before throwing the corresponding exception, this operator clears the provided object (it calls to clear() method) to leave it in a consistent state.
