class  DataSet
 A DataSet is an object that contains a set of n instances. Each instance contains a vector of m integers that represents the amount of money spent by a person in each one of m locations. This class uses a bidimensional matrix with n rows and m columns to store the values of the set of instances. This class also contains a vector of n labels (VectorInt _labels) to store the label of each instance. Each label is an integer that defines the class value of the corresponding instance. Finally, this class contains a vector of m Location objects (VectorLocation _locations) to store each locations in the DataSet. Classifies the instances of the dataset dataSetToClassify with the KNN algorithm and using the dataset dataSetModel as model to classify. Each instance in dataSetToClassify will be classified by changing its label to the one found by the KNN algorithm.

Exceptions
std::invalid_argumentThrows a std::invalid_argument exception if the number of locations in datasetModel and datasetToClassify is not equal.
std::invalid_argumentThrows a std::invalid_argument exception if doReductionDimensionality is true and K1 is less or equals than 0.
std::invalid_argumentThrows a std::invalid_argument exception if K2 is greater than the number of instances in datasetModel
std::invalid_argumentThrows a std::invalid_argument exception if K2 is less than 1
Parameters
dataSetModelDataSet used as model to classify. Input parameter
dataSetToClassifyDataSet to be classified. Input/output parameter
K1The K value to be used in the clustering algorithm (KMeans) of the dimensionality reduction step. Input parameter
K2The K value to be used in the KNN algorithm (to define the number K of nearest neighbours). Input parameter
doReductionDimensionalityIndicates whether or not dimensionality reduction should be applied using the clustering algorithm. Input parameter
