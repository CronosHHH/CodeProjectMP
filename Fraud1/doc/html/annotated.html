 CClusteringThis class is used to obtain a clustering of a given set of locations (Location objects). A clustering algorithm groups the input locations into K disjoint groups (clusters). The clustering algorithm used in this class is the K-Means algorithm. Each group will consist of a disjunct subset of the original location set and a representative location (centroid), which is calculated as the vector of the means of the locations in the group. The run() method in this class executes the algorithm.
