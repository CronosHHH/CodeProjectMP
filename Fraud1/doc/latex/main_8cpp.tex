{\footnotesize\ttfamily int main (\begin{DoxyParamCaption}\item[{int}]{argc,  }\item[{char \texorpdfstring{$\ast$}{*}}]{argv\mbox{[}$\,$\mbox{]} }\end{DoxyParamCaption})}

The purpose of this program is to read a set of locations from standard input, to cluster them into K clusters using the K-\/means algorithm, and to show the resulting clustering on the standard output.

This program first reads an integer K from standard input, which indicates the number of clusters to be used in the clustering process. Then, it reads a set of \doxylink{classLocation}{Location} objects from standard input. After reading the data, it performs the K-\/means clustering algorithm to group the locations into K clusters. Finally, it displays the resulting clustering information, including the cluster assignments for each location and the centroids of each cluster.

Be careful to show the output as in the below example.
