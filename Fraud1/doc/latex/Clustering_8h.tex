class \mbox{\hyperlink{classClustering}{Clustering}}
\begin{DoxyCompactList}\small\item\em This class is used to obtain a clustering of a given set of locations (\doxylink{classLocation}{Location} objects). A clustering algorithm groups the input locations into K disjoint groups (clusters). The clustering algorithm used in this class is the K-\/\+Means algorithm. Each group will consist of a disjunct subset of the original location set and a representative location (centroid), which is calculated as the vector of the means of the locations in the group. The \doxylink{classClustering_a17977c6467960744c8bc71ffb97aa9fc}{run()} method in this class executes the algorithm. An example of use is the following\+: \doxylink{classVectorLocation}{Vector\+Location} locations; Suppose here code to fill the vector of locations \doxylink{classClustering}{Clustering} clustering; clustering.\+set(locations, 5); clustering.\+run();. \end{DoxyCompactList}\end{DoxyCompactItemize}
